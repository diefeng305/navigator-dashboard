群晖配置说明：
1. 确保 DSM 已开启 Web 服务 (控制面板 → 网络 → DSM 设置 → 启用 HTTPS)
2. 端口默认为 5000 (HTTP) 或 5001 (HTTPS)
3. 使用 DSM 登录用户名和密码
飞牛配置说明：
1. 飞牛 OS 默认端口为 8000
2. 使用飞牛 Web 界面的登录用户名和密码
3. 需要飞牛 OS 版本支持 WebSocket API
威联通配置说明：
1. 确保 QTS 已开启系统监控服务
2. 默认端口为 8080 (HTTP) 或 443 (HTTPS)
3. 使用 QTS 登录用户名和密码
Unraid 配置说明：
1. 需要先在 Unraid 应用商店安装监控容器
2. 推荐安装 "UMA" 或 "unraid-simple-monitoring-api"
3. 安装后填写对应的端口地址 (默认 24940)
4. API Key 为可选，根据容器配置决定
